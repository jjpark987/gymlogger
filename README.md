# GymLogger

This repository contains the frontend for GymLogger.

## Geting Started

### Prerequisites

- Node.js
- npm
- Expo

### Installation

1. Clone the repository

```zsh
git clone git@github.com:jjpark987/gymlogger.git
```

2. Install dependencies

```zsh
npm install
```

3. Start the app

```zsh
npx expo start
```

Scan the QR code on an iPhone via Expo app to run.
